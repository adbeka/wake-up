# Wake-Up Clock

A simple web-based wake-up clock application that displays the current time and allows you to set an alarm to wake you up.

## Features

- Real-time clock display
- Set alarm time
- Alert notification when alarm time is reached
- Web interface accessible via browser

## How to Run

1. Install dependencies:
   ```
   npm install
   ```

2. Start the server:
   ```
   npm start
   ```

3. Open your browser and go to `http://localhost:3000`

4. Set your alarm time using the time input and click "Set Alarm"

5. When the current time matches the alarm time, an alert will pop up to wake you up.

## Clearing Alarm

Click "Clear Alarm" to remove the set alarm.
